Developer Handbook

Audience

This handbook is intended for Stardust developers.

Contents

The book is organized into the following topics and guides:

  • Modeling Guide: Introduction of modeling basics and integration issues.
  • Operation Guide: Description of the deployment to various application servers and administration facilities.
  • Programming Guide: Describes the basic concepts of the Stardust API and its relationship to the general process modeling and runtime concepts.
  • Deployment Guide: The Deployment Guide provides the information to deploy and configure a Stardust runtime environment.
  • Document Service Integration Guide
    The Document Service Integration provides the usage of the functionality for content repositories of document management systems.
  • Business Analysis and Reporting Component
    The Business Analysis and Reporting Component supports design and retrieval of arbitrary reports on business process data and data of business objects being accessed in these business processes. It is conceived to allow business analysts and developers to design and create ad-hoc or periodical reports either as printable material, as spreadsheets or as real-time views on business process data for monitoring purposes in scorecards or dashboards.
  • Simulation Feature
    The Simulation Feature allows a process model to be tested for performance characteristics without actually deploying the process.
  • Spring Integration Guide
    The Spring Integration Guide helps you to get started with using Stardust in a Spring environment.
  • Rapid Application Development
    The Rapid Application Development offers the possibility to get started quickly in application development without requiring a client-dependent development environment.
