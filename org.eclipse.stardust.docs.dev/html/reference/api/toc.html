Stardust API

The API documents contain the API specification of the Stardust's Application Programming Interface. The Stardust API is organized along the concepts of specific services needed to perform common usage scenarios.

  • Common API - API for common services.
  • Engine API - API for the engine services.
  • Web Services Engine API - API for the engine Web services.
  • Swing Portal API - API for the Stardust Swing Portal functionalities.
