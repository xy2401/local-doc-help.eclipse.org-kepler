Viewing the Results in the Predefined Report Monitoring Overview

In this chapter we will demonstrate how the security restriction set in the activity Activity by Scoped NationalDirector for reading activity instance data is reflected when monitoring the activity instances.

Open the Monitoring View

To view activity instance data in the Monitoring part of the Predefined Report Templates:

  1. Switch to the Business Control Center perspective.
  2. Select All Management Views.
  3. In the view tree, select Report Templates > Predefined Reports > Definition Overview.
  4. The Definition Overview opens as a separate view.
  5. Select the tab Monitoring

  6. In the Monitoring part, click on Monitor Active Activity Instances in the Activity Instances part.

  7. Click the Run Report icon in the toolbar:

  8. The parameter dialog opens. Here you have to adjust the EndTimeStamp entry to a date later than the current.

  9. Now the Monitoring - Active Activity Instances view is updated displaying information on completed activity instances.
