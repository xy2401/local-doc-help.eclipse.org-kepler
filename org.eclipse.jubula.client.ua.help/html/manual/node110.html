3.11 Working with Test Cases

Test Cases are the ''building blocks'' (keywords) for testing. They can:

  • contain other (referenced) Test Cases 3.9.2
  • contain Test Steps 3.16.1
  • be used in Test Suites 3.9.2
  • be used as Event Handlers 3.21
