4.1.1 Supported Swing AUT's

AUT's written with the Swing GUI toolkit are supported according to the following points:

  • The AUT is written using Java 1.4 or higher.
    Swing AUT's written in Java 1.4 cannot be started using an executable file 3.7.1 and cannot be started using the autrun option 3.7.2.6. For both of these option,s Java 1.5 or higher is required.
  • It uses a Java SE (Standard Edition) JRE.
    AUT's based on the NetBeans framework, and NetBeans itself, are currently not supported.
