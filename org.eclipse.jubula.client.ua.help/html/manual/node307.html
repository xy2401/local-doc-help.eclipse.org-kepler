3.33.1 Using Chronon when testing your AUT

You can configure your AUT to run with Chronon recording so that you can analyze any errors that occur during your automated test runs. See the section later 3.33.1.4 on analysing the reports for information on the tooling required to do this.

Jubula Feature users: The support for Chronon recording in AUT's is currently only available in the standalone software.
